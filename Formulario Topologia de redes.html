Prueba Formativa - Tipos de Redes de Computadoras

Pregunta 1: ¿Qué tipo de red se utiliza típicamente en un solo edificio o campus?

LAN (Local Area Network)
WAN (Wide Area Network)
MAN (Metropolitan Area Network)

Pregunta 2: ¿Cuál de las siguientes opciones describe mejor una red privada utilizada por una organización?

Internet
Intranet
Extranet

Pregunta 3: ¿Qué tipo de red se extiende sobre una amplia área geográfica, como un país o continente?

PAN (Personal Area Network)
CAN (Campus Area Network)
WAN (Wide Area Network)

Pregunta 4: ¿Qué protocolo se utiliza comúnmente para la transferencia de archivos en una red?

HTTP (Hypertext Transfer Protocol)
FTP (File Transfer Protocol)
SMTP (Simple Mail Transfer Protocol)

Pregunta 5: ¿Qué dispositivo se utiliza para interconectar redes y filtrar el tráfico basado en direcciones MAC?

Hub
Switch
Router
